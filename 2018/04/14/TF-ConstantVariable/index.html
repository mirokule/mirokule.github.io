Constant, Variable, Placeholder

发表于 2018-04-14
字数统计: 316

1. Constant

cons = tf.constant(value, dtype=tf.float32, shape=None, name=None, verify_shape=False)

参数

  • value: 初始值，必须是一个张量
  • dtype: 默认为value的数据类型
  • shape: 默认为value的shape
  • name: 常量名
  • verify_shape: 是否验证value的shape和指定shape是否相等，不相等时会抛出异常

2. PlaceHolder

placeholder也是一种常量，不同的是在运行时注入数据。
pla = tf.placeholder(dtype=tf.float32, shape=(), name=””)

3. Variable

变量Variable使用前需要初始化。
TensorFlow中有两个关于variable的构造函数：

  • tf.Variable(): 每次都创建新对象；创建时若命名冲突将覆盖原有对象
  • tf.get_variable(): 若命名对象存在则返回，否则创建新对象；创建时若命名冲突则报错

W1 = tf.Variable(initial_value=None, …)
W2 = tf.get_variable(name, initializer=None, …)
