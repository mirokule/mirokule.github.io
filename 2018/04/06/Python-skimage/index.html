Scikit-Image

发表于 2018-04-06
字数统计: 219

Scikit-Image是基于Scipy的图像处理包，将图片作为numpy数组处理。

子模块
子模块 功能
io 读写、显示图片
data 内置样本图片
color 颜色空间变换
filters 滤镜如图像增强、边缘检测
draw 图形绘制如线条、矩形
transform 几何变换如旋转、拉伸
morphology 形态学操作如骨架提取
exposure 强度调整如亮度、直方图均衡
feature 特征检测与提取
measure 图像属性测量如相似性、等高线
segmentation 图像分割
restoration 图像恢复
util 通用函数
