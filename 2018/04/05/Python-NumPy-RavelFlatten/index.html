NumPy数组降维

发表于 2018-04-05
字数统计: 327

NumPy中可以用于维度缩减的函数有：

  • ravel: 1-D,就地生效
  • flatten: 1-D,返回副本
  • squeeze: 移除shape为1的维度
  • reshape: 指定维度

1. ravel()

numpy.ravel(a, order=’C’)
将数组降维，返回一维数组；如果没有必要，不会产生原数据的副本，而是就地生效
相当于rashape(-1, order=order)

参数

  • a：输入array
  • order: {‘C’, ‘F’, ‘A’, ‘K’}，默认’C’
    • ‘C’: C-style，先处理行
    • ‘F’: Fortran-style，先处理列
    • ‘A’: Auto，如果a在内存中是Fortran contiguous则’F’，否则’C’
    • ‘K’: 按内存顺序

返回

  • y：1-D ndarray

2. flatten()

numpy.ndarray.flatten(order=’C’)
返回数组降维后的副本

参数

  • order: {‘C’,’F’,’A’,’K’}

返回

  • y: ndarray

3. squeeze()

numpy.squeeze(a, axis=None)
移除 single-dimensional entries，即shape中为1的维度。

参数

  • a: 输入array
  • axis: 指定要缩减的轴，若该轴的shape大于1，则报错

返回

  • squeezed: ndarray

实例

4. reshape()

numpy.reshape(a, newshape, order=’C’)
为数组指定新的shape

参数

  • a: 输入array
  • newshape: int表示该维度长度，-1表示自动推断
  • order: {‘C’, ‘F’, ‘A’}

返回

  • reshaped_array: ndarray
