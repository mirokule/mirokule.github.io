pandas.DataFrame

发表于 2018-05-19
字数统计: 449

class pandas.DataFrame(data=None, index=None, columns=None, dtype=None, copy=False)

属性

数据容量

  • ndims: 维度
  • shape: 尺寸(h,w,b)
  • size: 元素个数=h×w×b
  • dtypes: 元素类型

表格组成

  • axes: 横纵轴的列表
  • index: 行标签
  • columns: 列标签
  • values: 值的numpy数组

数据选取

  • at: 根据行列名称选取单个元素
  • iat: 根据行列序号选取单个元素
  • loc: 根据行列名称\Boolean序列选取内容
  • iloc: 根据行列序号选取内容

其他

  • T: 转置
  • empty: 是否为空

方法

重复数据

  • duplicated(): 返回boolean序列
  • drop_duplicates()

缺失值

  • isna(), isnull()
  • notna(), notnull()
  • dropna()
  • fillna()

添加删除

  • append(): 添加行
  • insert(): 插入列
  • drop(): 删除行或列

替换更新

  • replace(): 用新值替换旧的
  • update(): 用另一个dataframe中的元素更新

重命名行列标签

  • rename(), rename_axis()
  • add_prefix(), add_suffix(): 给标签添加前缀\后缀

查看内容

  • head(n), tail(n)
  • info(): 简明摘要
  • describe(): 统计信息

统计函数

  • sum()
  • prod(), product()
  • std(), var()
  • skew(), kurt(), kurtosis()
  • max(), min(), mean(), mode(), median()

单表运算

  • clip(), clip_lower(), clipupper()
  • round()
  • abs()

双表运算

  • add(): 两个dataframe的元素相加
  • sub(), subtract()
  • dot(): 矩阵乘法
  • mul(), multiply(): 元素乘法
  • div, divide()
  • mod(): 取模
  • corrwith(): 相关性

双表合并

  • merge()
  • join()
  • combine()
  • combine_first()

重排

  • squeeze()
  • stack()
  • unstack()
  • pivot
  • pivot_table()

排序抽样

  • sort_index()
  • sort_values()
  • sample(n): 抽样样本
  • nlargest(n), nsmallest(n): 根据指定列排序后，返回最大/最小的n行

迭代器

  • items(), iteritems(): (
  • iterrows()

绘图

  • plot()
  • boxplot()
  • hist()

存储

  • to_csv()
  • to_excel()
  • to_hdf()
