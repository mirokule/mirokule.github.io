pandas.read_csv

发表于 2018-05-18
字数统计: 286

参数

  • filepath_or_buffer: 文件路径
  • sep: 分隔符，默认’,’，正则’\r\t’
  • delimiter: sep的别名
  • header: 指定行数作为列名，header=0则从第一行推断
  • names: 外部传递的列名，对应header=None
  • squeeze: 如果文件只包含一列，则返回一个Series
  • prefix: 没有列标题时，给列标题添加前缀
  • dtype: 每列数据的数据类型
  • nrows: 需要读取的行数
  • thousands: 千分位分隔符
  • decimal: 小数点
  • encoding: 字符集编码类型

分块读取大文件

  1. 通过参数chunksize指定文件分块大小，返回可迭代的对象TextFileReader

  2. 指定iterator=True也可以返回一个TextFileReader
