使用后端框架的激活函数

高级激活函数

对于后端Theano/TensorFlow/CNTK不能表达的复杂激活函数，如PReLU, LeakyReLU等，可以通过高级激活层(Advanced Activation)实现。
注意此时是在构建layer对象。

LeakyReLU层

LeakyReLU是ReLU的特殊版本，当不激活时仍有非零输出值，从而获得一个小梯度，避免神经元“死亡”现象。
$$f(x)=
\begin{cases}
\alpha\cdot x& \text{x < 0}\\
x& \text{x >= 0}
\end{cases}$$

参数alpha: float >= 0，代表第三象限中线段的斜率。

PReLU层

参数化的ReLU(Parametric ReLU)，表达式为：
$$f(x)=
\begin{cases}
\alpha\cdot x& \text{x < 0}\\
x& \text{x >= 0}
\end{cases}$$

其中参数 alpha 是通过学习得到的，和X具有相同形状。

  • alpha_initializer: alpha的初始化函数
  • alpha_regularizer: alpha的正则项
  • alpha_constraint: alpha的约束项
  • shared_axes: 该参数指定的轴将共享同一组参数
ELU层

指数线性单元(Exponential Linear Unit)，表达式为：
$$f(x)=
\begin{cases}
\alpha(e^x - 1)& \text{x < 0}\\
x& \text{x >= 0}
\end{cases}$$

ThresholdedReLU层

带有门限的ReLU，表达式为：
$$f(x)=
\begin{cases}
x& \text{x > theta}\\
0& \text{otherwise}
\end{cases}$$
