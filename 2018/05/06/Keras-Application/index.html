Keras预训练模型Application

发表于 2018-05-06
字数统计: 558

应用模块Application提供了带有预训练权重的模型，可以用来预测和特征提取。

Xception, MobileNet 模型仅在TensorFlow下可用，其他的兼容Theanno和TensorFlow。

Xception

默认输入图片尺寸为299×299。

  • include_top: 是否保留顶层的3个全连接网络
  • weights: ‘imagenet’预训练权重, ‘None’随机初始化权重
  • input_tensor: Keras tensor作为图片输入
  • input_shape: 仅当include_top=False时有效，指明输入图片的尺寸
  • pooling: 当include_top=False时指定池化方式，{‘None’, ‘avg’, ‘max’}
  • classes: 当include_top=True并且不加载权重时可用，指定图片分类的类别数

MobileNet

默认输入图片尺寸224×224.

  • alpha: 控制网络的宽度，>1同比例增加滤波器，<1同比例减少，=1默认
  • depth_multiplier: depthwise卷积的深度乘子
  • dropout: dropout比例

VGG16, VGG19, ResNet50

默认输入图片尺寸为224×224

ResNet50

默认输入图片尺寸为224×224

InceptionV3, InceptionResNetV2

默认输入图片尺寸为299×299
