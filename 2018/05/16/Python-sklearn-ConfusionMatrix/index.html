混淆矩阵ConfusionMatrix

发表于 2018-05-16
字数统计: 336

Confusion Matrix用于直观显示多分类问题中，分类结果的错误分布。

图中第(i,j)个单元格表示类别i的样本被学习器分类到类别j的数量。

API

sklearn.metrics.confusion_matrix(y_true, y_pred, labels=None, sample_weight=None)
参数

  • y_true: 真实值
  • y_pred: 预测值
  • labels: 标签
  • sample_weight: 样本权重

返回

  • C: array, shape=[n_classes, n_classes]
