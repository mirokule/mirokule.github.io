发表于 2018-02-28
字数统计: 432

定义

参数说明：

  • data: 矩阵数据集，可以是Numpy的Array，也可以是pandas的DataFrame
  • vmin, vmax: 锚定颜色的最大值和最小值(values to anchor the colormap)
  • cmap: matplotlib的colormap名称或颜色对象，用以调整颜色类别
  • center: 颜色中的均值数据(the value at which to center the colormap)，用以调整图像颜色的深浅
  • robust: 如果为True且vmin、vmax没有设置，那么颜色将根据强分位数计算，而不是根据极值计算
  • annot: 单元格上的注释(annotation)，为True则填充数据；为注释
  • fmt: 注释的字符串格式(formatting)
  • annot_kws: 注释的字典参数
  • linewidths: 单元格边框宽度
  • linecolor: 单元格边框颜色
  • cbar: 是否显示colorbar
  • cbar_kws: fig.colorbar的字典参数，如{‘orientation’:’horizontal’}
  • cbar_ax: 显示colorbar的轴域(axes)
  • square: 为True则单元格画成正方形
  • xticklabels: 列名，默认True，也可以是list-like或int
  • yticklabels: 行名，默认Ture，也可以是list-like或int
  • mask: 为True的位置不显示单元格
  • ax: 绘图的轴域(axes)
  • **kwargs: 其他参数

函数返回：

  • ax: matplotlib Axes
