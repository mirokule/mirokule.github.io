1.表面上看，IPython就像是一个增强的交互式Python解释器，可以使用Tab键自动完成。
在变量的前面或后面加上一个问号(?)就可以将有关该对象的一些通用信息显示出来，使用双问号(??)还可以显示出该函数的源代码。

3.IPython Notebook是一个运行在命令行上的轻量级服务器进程，使用基于JSON的文档格式.ipynb，可以轻松分享代码、输出结果以及图片等内容。

快捷键

  • Ctrl-P 后向搜索命令历史
  • Ctrl-N 前向搜索命令历史
  • Ctrl-Shift-V 从剪贴板粘贴文本
  • Ctrl-C 中断当前执行的代码
  • Ctrl-A 将光标移动到行首
  • Ctrl-E 将光标移动到行尾
  • Ctrl-F 将光标向前移动一个字符
  • Ctrl-B 将光标向后移动一个字符
  • Ctrl-K 清除从光标开始到行尾
  • Ctrl-U 清除当前行
  • Ctrl-L 清屏

魔术命令

魔术命令(Magic Command)是以百分号(%)为前缀的命令，为常见任务提供便利。
使用问号(?)同样可以查看命令选项。

  • %run 可以执行一个py脚本
  • %time 执行一条语句，报告执行时间
  • %timeit 多次执行同一语句，报告执行时间
  • %debug 进入调试器
  • %pdb 在异常发生后自动进入调试器
  • %paste 执行剪贴板中的代码
  • %cpaste 打开一个特殊提示符以便手工粘贴待执行的Python代码
  • %hist 打印命令的输入历史
  • %who、%whos、%who_ls 显示命名空间中定义的变量
  • %reset 删除命名空间中的全部变量
  • %quickref 显示IPython的快速参考
  • %magic 显示所有魔术命令的详细文档

IPython的另一个重要特点是跟操作系统shell结合得非常紧密，可以在IPython中直接使用Linux系统命令，而不用像Python那样import os。调用shell命令的魔术命令：

  • %pwd 返回当前工作目录
  • %cd directory 更改当前工作目录
  • %pushd directory 将当前目录入栈，并转向目标目录
  • %popd 弹出栈顶目录，并转向该目录
  • %alias alias_name cmd 为系统shell命令定义别名
  • !cmd 在系统shell中执行cmd
