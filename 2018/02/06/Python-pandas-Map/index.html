Series.Map

Series.map(arg, na_action=None)
Map是Series特有的方法，DataFrame中没有。
不在字典里的值将被转成NaN，或者_missing_中预设的值。

参数

  • arg: function, dict, or Series
  • na_action: {None, ‘ignore’}，转换时是否忽略NA

返回

  • y: Series

实例

Series.apply

Series.apply(func, convert_dtype=True, args=(), **kwds)
在Series上应用函数，可以是NumPy ufunc(作用于整列)，也可以是Python function(作用于元素)。

参数

  • function: function
  • convert_dtype: 默认True，寻找更好的dtype；若为False则dtype=object
  • args: 传递给function的其他参数
  • kwds: keyword参数

返回

  • y

实例

DataFrame.apply

DataFrame.apply(func, axis=0, broadcast=False, raw=False, reduce=None, args=(), **kwds)
作用于行列

参数

  • func: function
  • axis: 默认0，作用于每列；若为1则作用于每行
  • broadcast: 默认False
  • raw: 默认False，将每行列转为Series; 若为True则接受ndarray作为输入
  • reduce
  • args

返回

  • applied

实例

DataFrame.applymap

DataFrame.applymap(func)
为每个元素传递函数变换，像作用于Series的Map函数一样。

参数

  • func: function

返回

  • applied: DataFrame
