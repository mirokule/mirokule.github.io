seaborn.distplot

发表于 2018-03-14
字数统计: 198

结合了matplotlib的hist函数和seaborn的kdeplot()、rugplot()。

参数：

  • a: 一维数组, Series, list
  • bins: 条形宽度
  • hist: 是否绘制柱状图
  • kde: 是否绘制Gaussian Kernel Density Estimate
  • rug: 是否在support label上绘制rugplot(小竖条)
  • fit: 用于对比的标准分布
  • {hist, kde, rug, fit}_kws: 关键词定义的参数字典
  • color: 绘图颜色
  • vertical: 若为True，则纵向绘制
  • norm_hist: 使用密度而不是数量绘图
  • axlabel: name for support axis label
  • label: legend label
  • ax: matplotlib axes

返回：

  • ax: matplotlib axes
