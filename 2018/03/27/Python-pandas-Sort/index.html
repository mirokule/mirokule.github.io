1.sort_values

DataFrame.sort_values(by, axis=0, ascending=True, inplace=False, kind=’quicksort’, na_position=’last’)
对某列或某几列values排序，返回排序后的DataFrame.

参数

  • by: str, list, 排序依据的列名
  • axis: 默认0对行排序，1对列
  • ascending: 默认True升序
  • inplace: 默认False返回拷贝
  • kind: {‘quicksort’,’mergesort’,’heapsort’}排序算法
  • na_position: {‘first’, ‘last’}，把NaN放在开头还是末尾

返回

  • sorted_obj: DataFrame

实例

2.sort_index

DataFrame.sort_values(axis=0, level=None, ascending=True, inplace=False, kind=’quicksort’, na_position=’last’, sort_remaining=True, by=None)

参数

  • axis, ascending, inplace, kind, na_position: 同上
  • level: 多层索引中指定index level
  • sort_remaining: 多层索引中，对指定level排序后，继续对剩余的index排序

返回

  • sorted_obj: DataFrame

3.nlargest

DataFrame.nlargest(n, columns, keep=’first’)
返回排序后最大的N个样本(整行)。

参数

  • n: 返回值的个数
  • columns: str, list，排序依据的列名
  • keep: {‘first’,’last’}若有重复值，取开头的还是末尾的

返回

  • return: DataFrame

实例
