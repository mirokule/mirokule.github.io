对指定参数的穷举搜索。

参数

  • estimator: 需要实现sklearn estimator interface
  • param_grid: 要调校的参数名称及可选值
  • scoring: 评分策略
  • fit_params: 废弃
  • n_jobs: 并行任务数
  • pre_dispatch: 并行过程中分发的任务数？
  • iid: identically distributed
  • cv: int, cross-validation generator
  • refit: 使用得到的最优参数，重新训练模型
  • verbose: 输出信息冗余度
  • error_score: 错误处理
  • return_train_score: 是否返回训练集得分
