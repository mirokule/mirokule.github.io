sklearn.linear_model

发表于 2018-03-04
字数统计: 183

名词解析

  • parameters(参数)：参变量，描述某种特征值。
  • coefficients(系数)：描述某种关系，比如两个变量之间的变化关系。

LinearRegression

参数：

  • fit_intercept: 默认True，是否计算截距b
  • normalize: 默认False，是否对X作l2规范化
  • copy_X: 默认True，若为False将修改原数据
  • n_jobs: 并发任务数

属性：

  • coef_: 系数数组A
  • intercept_: 截距b

方法：

  • fit(X, y, sample_weight=None): 训练模型，可以带权重
  • score(X, y, sample_weight=None): 验证并计算得分($1- \frac{u}{v}$，其中u是残差平方和，v是y的方差)，最佳分数为1.0
  • predict(X): 预测并返回结果
  • set_params(**params): 设定参数
  • get_params(deep=True): 获取参数
