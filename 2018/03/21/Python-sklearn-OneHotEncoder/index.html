sklearn.OneHotEncoder

发表于 2018-03-21
字数统计: 238

LabelEncoder将不同类别转换为数字，比如[‘paris’,’tokyo’]转为[2,3]，引入了类别大小属性。
OneHotEncoder将类别编码成m个二值类别，只标示类别，没有引入大小。
比get_dummies的优势在于可以处理数字类别。

定义

class sklearn.preprocessing.OneHotEncoder(n_values='auto',categorical_features='all',dtype=<class'numpy.float64'>,sparse=True,handle_unknown='error')

参数

  • n_values: 特征的类别数量，’auto’由训练数据计算，也可以指定int
  • categorical_features: 指定要转换的字段
  • dtype: 输出的类型
  • sparse: 默认True返回系数矩阵，若为False返回Array
  • handle_unknown: 是否raise error

属性

  • active_features_: 训练集中的被转换的values
  • feature_indices_: indices to feature ranges
  • n_values_: 每个特征的number of values

方法

  • fit(X, y=None)
  • fit_transform(X, y=None)
  • transform(X)
  • get_params([deep])
  • set_params(**params)
