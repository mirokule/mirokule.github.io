Octave常用命令

发表于 2018-01-12
字数统计: 444

符号%

Octave和Matlab代码注释使用符号 %

数组索引从 1 开始

帮助

help command // 可以查询command的详细用法

读写数据

pwd // 显示当前path
cd NewPath // 更改路径

load filename // load('filename')
save filename v // 将变量v的数据写入文件
who
whos // 更多详细信息
clear v // 删除变量

生成矩阵

A = [1,2;2,3;4,5]
v = [1 2 3 4 5] // 行向量
v = [1;2;3;4;5] // 列向量
v = 1:0.5:5 // 按步长增加

B = ones(3,3)
B = zeros(3,3)
B = rand(2,2) // 随机数
B = randn(1,3) // 随机整数    
I = eye(3) // 单位矩阵

sz = size(A)
len = length(v) // 最长的维度
A(3,2) // 元素$A_{32}$
A(3,:) // 第3行所有元素
A([1 3],:) // 第1、3行的所有元素
A(:) // 将所有元素变成一列
A = (A, v) // 在A矩阵右侧增加一列
v(1:10)

矩阵运算

A*B // 矩阵相乘
A.*B // 元素乘法
A.^2 // 元素平方

A' // 转置$A^T$
pinv(A) // 逆矩阵

v + ones(length(v),1) = v + 1
[value, index] = max(v) // 返回最大值和索引
v < 0 // 逐个元素判断，返回1/0
find(v<0) // 返回正例索引

sum(v) // 元素和
prod(v) // 元素积
sum(A, DIM) // 制定运算维，1行相加得到各列和，2列相加得到各行和

控制语句

for i=1:10
    v = i+1;
end

while true
    i += 1
    if i == 6
        break
    end
end

函数

function [y1, y2] = addNumber(x)
y1 = x + x
y2 = x * 3

// 调用，需指定路径
[a,b] = addNumber(111)

绘图

plot(X,Y,'r')
axis([1 2 3 4])
title('name')
xlabel('axis')
legend('sin','cos')

figure(2) // 新建图像
subplot(2,2,1) // 绘制子区域

scatter(X,Y)
hist(w,nBINs)

imagesc(5) // 彩色矩阵图
colorbar
colormap gray

hold on // 保持现有图线，不会被后续覆盖
clf // 清除图像
close 

print -dpng 'picName.png'
# Octave
