线性方程组和矩阵

发表于 2018-01-24
字数统计: 1,165

线性方程组

A linear system is a collection of one or more linear equations involving the same variables. A linear system has

  • no solutions
  • exactly one solution
  • infinitely many solutions

方程$Ax=b$有解当且仅当$b$是$A$的各列的线性组合。
每个线性方程代表高维空间的一个超平面，线性方程组的解是所有超平面的交点。
解方程组的基本思路是把方程组用一个更容易解的等价方程组代替。
若两个线性方程组的增广矩阵(augmented matrix)是行等价的，则它们具有相同的解集(solution set)。

阶梯形矩阵

一个矩阵称为阶梯形(echelon form)(或行阶梯形)，若它有以下三个性质：

  • 每一非零行在每一零行之上；
  • 某一行的先导元素(leading entry)所在的列位于前一行先导元素的右面；
  • 某一先导元素所在列下方元素都是零。
    若一个阶梯形矩阵还满足以下性质，称为简化阶梯形(reduced echelon form)：
  • 每一非零行的先导元素是1；
  • 每一先导元素1是该元素所在列的唯一非零元素。

定理
每个矩阵行等价于唯一的简化阶梯形矩阵。
主元位置
矩阵中的主元位置(pivot position)是A中对应于它的阶梯形中先导元素的位置。主元列是A的含有主元位置的列。

线性方程组的解

对应主元列的变量称为基本变量，其他变量称为自由变量。
齐次方程$Ax=0$有非平凡解(非零向量)，当且仅当方程至少有一个自由变量。

线性无关

$\mathbb{R}^n$中一组向量{$\vec{v}_1,…\vec{v}_p$}称为线性无关的，若向量方程
$$ x_1\vec{v_1} + x_2\vec{v_2} +\cdot\cdot\cdot+ x_p\vec{v_p} = 0 $$
仅有平凡解。向量组称为线性相关的，若存在不全为零的权$c_1,…c_p$，使得
$$ c_1\vec{v_1} + c_2\vec{v_2} +\cdot\cdot\cdot+ c_p\vec{v_p} = 0 $$
矩阵A的各列线性无关，当且仅当$Ax=0$仅有平凡解。
定理
若一个向量组的向量个数超过每个向量元素个数，那么这个向量组线性相关。即$\mathbb{R}^n$中任意向量组{$\vec{v_1},…,\vec{v_p}$}，当$p>n$时线性相关。

矩阵

矩阵AB中的每一列，都是A中列向量的线性组合。
$(AB)^T = B^TA^T$
$(AB)^{-1} = B^{-1}A^{-1}$
$(A^T)^{-1} = (A^{-1})^T$
若A可逆，它的逆矩阵是唯一的，记做$A^{-1}$，可逆矩阵称为非奇异矩阵；不可逆矩阵称为奇异矩阵。

可逆矩阵定理

设A为n*n矩阵，则下列命题等价，同时为真或同时为假：

  • A是可逆矩阵
  • A等价于n*n单位矩阵
  • A有n个主元位置
  • A的各列线性无关
  • 方程$Ax=0$仅有平凡解
  • 对$\mathbb{R}^n$中任意b，方程$Ax=b$至少有一个解
  • A的各列生成(span)$\mathbb{R}^n$(满秩)
  • 线性变换$x\mapsto Ax$是一对一的
  • 线性变换$x\mapsto Ax$把$\mathbb{R}^n$映射到$\mathbb{R}^n$上
  • 存在n*n矩阵C使得$CA=I$
  • 存在n*n矩阵D使得$AD=I$
  • $A^T$是可逆矩阵

初等矩阵

把单位矩阵进行一次行变换，得到初等矩阵E，E是可逆的，E的逆矩阵能把E变回I。
矩阵EA表示对矩阵A进行某种初等行变换。
把增广矩阵$\left[A\; I\right]$进行行化简，得到$\left[I\;A^{-1}\right]$，求得A的逆矩阵。

LU分解

矩阵A可以分解为一个单位下三角矩阵(L)和一个上三角矩阵(U)的乘积。

解空间

矩阵A的列空间是A的各列的线性组合的集合，记做Col A.
矩阵A的零空间是齐次方程$Ax=0$的所有解得集合，记做Nul A.
非零子空间H的维数，是H的任意一个基的向量个数，记做dim H.
矩阵A的秩是A的列空间的维数，记做rank A.
如果一个矩阵A有n列，则$rank\;A + dim\;Nul\;A = n$.

行列式

行列式是一个数，记做$\Delta$。
方阵A的行列式具有以下性质：

  • 若A的某一行的倍数加到另一行得到矩阵B，则$det\;B=det\;A$
  • 若A的两行互换得到矩阵B，则$det\;B=-det\;A$
  • 若A的某行乘以k倍得到矩阵B，则$det\;B=k\cdot det\;A$

方阵A是可逆的当且仅当$det\;A\neq 0$.
若A是一个n阶矩阵，则$det\;A^T=det\;A$.
若A和B均为n阶矩阵，则$det\;AB=(det\;A)(det\;B)$.
