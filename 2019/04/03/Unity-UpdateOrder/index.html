Upate, FixUpdate, LateUpdate

发表于 2019-04-03
字数统计: 128
FixedUpdate

固定时间间隔更新，与帧率无关。
如果帧率很低，可能一帧内更新多次；如果帧率很高，可能多帧才更新一次。
在FixedUpdate内计算运动轨迹时不需要乘以时间间隔Time.deltaTime.

Update

每帧执行一次，更新的主力函数。

LateUpdate

每帧执行一次，在Update函数完成之后。
常见应用包括第三人称摄像机的位置和转矩更新，以确保Update中角色的移动已经完成。
