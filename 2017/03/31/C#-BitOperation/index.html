异或

异或：两者相同为0，两者不同为1；命题为 “两者的值不同”，“有且仅有一个为真”。
用途：
(1)快速比较两个值： a ^ b == 0
(2)翻转特定位置：与00100按位异或将使原数字的第3位翻转
(3)判断一个二进制数中1的数量是奇数还是偶数
(4)不使用其他空间，交换两个值
