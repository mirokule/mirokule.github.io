Unity-Collision

发表于 2016-09-27
字数统计: 116

碰撞

物体发生碰撞的条件

  • 两个物体都必须有碰撞器(Collider)
  • 至少一个物体有刚体(Rigidbody)

两个物体都不勾选isTrigger，进入OnCollisionEnter
两个物体中至少有一个勾选isTrigger，进入OnTriggerEnter，不进入OnCollisionEnter

API

触发(is Trigger)

  • OnTriggerEnter(Collider collider)
  • OnTriggerExit(Collider collider)
  • OnTriggerStay(Collider collider)

碰撞

  • OnCollisionEnter(Collision collision)
  • OnCollisionExit(Collision collision)
  • OnCollisionStay(Collision collision)

Collision

  • collider: the collider we hit
  • contacts: contact points
  • gameObject
  • impulse
  • relativeVelocity
  • rigidbody: the rigidbody we hit. null if the collider has no rigidbody attached.
  • transform

Dot 点积

u.v = |u||v|cos<u,v> = ux * vx + uy * vy，得到标量

在计算机图形学和游戏编程中，点积的重要用途之一是计算向量在给定方向上的分量(投影)。

Proj u = (u.v) * v / (|v|*|v|)

另外用于计算夹角
点积大于0则两个向量的夹角小于90度,锐角; 点积等于0，直角; 点积小于0,钝角

Vector3 a = new Vector3(1,1,1);
Vector3 b = enw Vector3(1,5,1);

float result = Vector3.Dot(a,b);
float angle = Vector3.Angle(a,b); //夹角，返回[0 - 180]

Cross 叉积

仅当向量包含3个或更多分量时，叉积才有意义。

u*v = |u|*|v|*sin<u,v> * n

其中n是uv平面的单位法向量。
实际应用中，使用矩形形式计算单位法向量n，很少直接使用叉积。

// c = a x b，得到向量，向量的模为 |c| = |a||b|sin<a,b>，且向量c垂直于a、b组成的平面，a x b = - b x a(左/右手定则?)
Vector3 c = Vector3.Cross(a,b);

// 判断顺时针、逆时针方向，指定XZ平面，考察叉积的Y轴分量
if (c.y > 0)
{ ... }

// 返回夹角，给angle标明方向
float sign = Mathf.Sign(Vector3.Dot(c.normalized, Vector3.Cross(a.normalized, b.normalized)));
float signed_angle = angle * sign;
