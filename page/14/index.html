SpriteKit-Anchor Point and Position

发表于 2015-10-13
字数统计: 80

Position

A node’s position is its location in parent coordinate system.

Anchor Point

??A skscene‘s anchorPoint property determins the origin point in the scene.

A sprite node’s anchorPoint property determines which point in the frame is positioned at the sprite’s position.

Example

  1. Point = (80, 80), anchorPoint = (0.5, 0.5): the sprite puts its center on (80,80)

  2. Point = (80, 80), anchorPoint = (0, 1): the sprite puts its left-top point on (80,80)

Hashable

发表于 2015-09-14
字数统计: 114

Update

HashValue is used to group objects in different bins, in which they are compared to one another using “==” method.
Therefore, hashValue will always be called when inserting an object into a set. It ensures the bins are as small as possible, minimizing calls to “==”.

In order to compare objects or put objects in Set

class ClassName: Hashable {
    ...

    var hashValue: Int {
        return NUM
    }
}

And also, hashable should conform to Equatable, as follows:

protocol Hashable: Equatable {
    var hashValue: Int { get }
}

protocol Equatable {
    func ==(lhs: Self, rhs: Self) -> Bool
}

So, add == function OutSide class

func ==(lhs: Point, rhs: Point) -> Bool {
    return lhs.column == rhs.column && lhs.row == rhs.row
}
