CMMotionManager

发表于 2016-05-19
字数统计: 147

A CMMotionManager is the gateway to the motion services provided by iOS.

An app should create only a single instance of the CMMotionManager class.

Data Type

Accelerometer

accelerometerData.acceleration(x: Double, y: Double, z: Double)

Gyroscope

rotationRate(x: Double, y: Double, z: Double): raw data

Magnetometer

magnetometerData.magnetcField

DeviceMotion

  • attitude: CMAttitude, 朝向
  • rotationRate: bias removed
  • gravity
  • userAcceleration
  • mangeticField: device bias removed

Data Fetch

Pull: periodic sampling

Takes no arguments.
Recommended for games, for most games are interested only the lastest motion data.

Push

Takes an operation queue and a block handler for processing updates.

Must call stop when you no longer want your app to process updates.

SKCameraNode

发表于 2016-05-19
字数统计: 98

An SKCameraNode is used to specify a position in the scene that the scene can be rendered from.
If a camera node is placed in the scene and the scene’s camera property is set to the camera node, then the scene is rendered using the camera node’s properties.

The scene is rendered so that the camera node’s origin is placed in the middle of the scene.

let cameraNode = SKCameraNode()
addChild(cameraNode)
camera = cameraNode
cameraNode.position = CGPoint(x: self.size.width / 2, y: self.size.height / 2)

Use camera node instead of moving background node manually.

SceneKit Force and Torque

字数统计: 70

Force

Can affect linear and angular momentum.

    let force = SCNVector3(x: randomX, y: randomY, z: 0)
    let position = SCNVector3(x: 0.05, y: 0.05, z: 0.05)
geometryNode.physicsBody?.applyForce(force, atPosition: position, impulse: true)

Torque

Only affects angular momentum.

let torque = SCNVector4(x: 0.1, y: 0.1, z: 0.1, w: 1.0)

The w-componet determines the rotation angle, that is, the magnitude of the torque.

Elements of color

  1. HSB，对应眼睛感知
  • Hue: 色调、色相，光的波长
  • Saturation: 饱和度，鲜艳度
  • Brightness: 明度、亮度，明暗

  • Green: 绿
  • Blue: 蓝

  • Magenta: 品红 = 蓝+红
  • Yellow: 黄 = 红+绿

取值： 0~100%

The CGAffineTransform data structure represents a MATRIX used for affine transformations.
A transformation specifies how points in one coordinate system map to points in another coordinate system.

Creating

  • CGAffineTransformMakeRotation(_ angle: CGFloat) -> CGAffineTransform
  • CGAffineTransformMakeScale(_ sx: CGFloat, _ sy: CGFloat) -> CGAffineTransform
  • CGAffineTransformMakeTranslation(_ tx: CGFloat, _ tx: CGFloat) -> CGAffineTransform: move by axis

Modification

  • CGAffineTransformTranslate(_ t: CGAffineTransform, _ tx: CGFloat, _ ty: CGFloat) -> CGAffineTransform: move the transform
  • CGAffineTransformScale
  • CGAffineTransformRotate
  • CGAffineTransformInvert(_ t: CGAffineTransform) -> CGAffineTransform
  • CGAffineTransformConcat(_ t1: CGAffineTransform, _ t2: CGAffineTransform) -> CGAffineTransform: combine two transform

Apply

  • CGRectApplyAffineTransform(_ rect: CGRect, _ t: CGAffineTransform) -> CGRect: apply the transform to rectangle

Evaluating

  • CGAffineTransformIsIdentity(- t: CGAffineTransform) -> Bool
  • CGAffineTransformEqualToTransform(_ t1: CGAffineTransform, _ t2: CGAffineTransform) -> Bool

CATransaction

发表于 2015-11-01
字数统计: 102

CATransaction is the Core Animation mechanism for batching multiple layer-tree operations into atomic updates to the render tree.

  • Implicit: any modification to CALayer without active transactions, automaticlly comitted
  • Explicit: active transactions, need begin() and commit()

Creatint and Committing

  • begin: begin a new transaction for the current thread
  • commit: commit all changes made during the current transaction
  • flush: flush any extant implict transaction, called automaticlly at the end of current runloop

Disable Animation

  • disableActions: whether actions are suppressed
  • setDisableActions

Duration and Timing

  • animationDuration
  • setAnimationDuration
  • animationTimingFunction: returns CAMediaTimingFunction
  • setAnimationTimingFunction

Lock

  • lock: prevent modification from other threads
  • unlock
