CATransaction

发表于 2015-11-01
字数统计: 102

CATransaction is the Core Animation mechanism for batching multiple layer-tree operations into atomic updates to the render tree.

  • Implicit: any modification to CALayer without active transactions, automaticlly comitted
  • Explicit: active transactions, need begin() and commit()

Creatint and Committing

  • begin: begin a new transaction for the current thread
  • commit: commit all changes made during the current transaction
  • flush: flush any extant implict transaction, called automaticlly at the end of current runloop

Disable Animation

  • disableActions: whether actions are suppressed
  • setDisableActions

Duration and Timing

  • animationDuration
  • setAnimationDuration
  • animationTimingFunction: returns CAMediaTimingFunction
  • setAnimationTimingFunction

Lock

  • lock: prevent modification from other threads
  • unlock

  • kCAMediaTimingFunctionEaseIn
  • kCAMediaTimingFunctionEaseOut

CGGeometry

发表于 2015-10-31
字数统计: 192

CGGeometry Reference defines structures for geometric primitives and functions that operate on them, including CGPoint, CGSize, and CGRect.
For example, a rectangle with an origin of [0.0, 0.0] and a size of [10.0,10.0] is exactly equivalent to a rectangle with an origin of [10.0, 10.0] and a size of [-10.0,-10.0].

Create

  • CGPointMake // same as CGPoint(x:y:)
  • CGRectMake
  • CGSizeMake
  • CGVectorMake

Compare

  • CGPointEqualToPoint
  • CGSizeEqualToSize
  • CGRectEqualToRect
  • CGRectIntersectsRect

Membership

  • CGRectContainsPoint
  • CGRectContainsRect

Get Property

  • CGRectGetMinX
  • CGRectGetMidY
  • CGRectGetHeight

Modify Rectangles

  • CGRectDivide: divide a source rectangle into two component rectangles
  • CGRectIntersection: return the intersection of two rectangles

  • CGRectUnion: return the smallest rectangle that contains the two source rectangles

  • CGRectInset: return a rectangle that is smaller or larger than the source rectangle, with the same center point

  • CGRectOffset: return a rectangle with an origin that is offset from that of the source rectangle

  • CGRectIntegral: return the smallest rectangle that results from converting the source rectangle values to integers

  • CGRectStandardize: return a rectangle with a positive width and height

Constants

  • CGRectInfinite
  • CGRectNull
  • CGPointZero, CGRectZero, CGSizeZero
  • CGRectEdge: MinXEdge, MaxYEdge…(MinY is top in iOS while bottom in OS X and SpriteKit)
