  • rotation_range: 随机旋转的角度
  • width_shift_range: 水平偏移的幅度
  • height_shift_range: 竖直偏移的幅度

  • fill_mode: ‘constant’,’nearest’, ‘reflect’, ‘wrap’超出边界的点的处理方式

Keras序贯模型Sequential

发表于 2018-05-05
字数统计: 983

序贯模型是多个网络层的线性堆叠。

快速开始

构造
可以通过传递一个layer的list来构造模型，也可以通过add()方法将Layer逐个加入。

输入
Sequential的第一层需要接受一个输入数据shape的参数，后面的各个层可以自动推导出中间数据的shape.
为第一层指定输入数据shape的方法包括：

  • input_shape: tuple类型，不包括batch大小
  • input_dim: 某些层支持该参数
  • batch_size: 某些层需要指定固定大小的batch_size

编译
训练模型之前，需要通过compile对模型编译。
compile(self, optimizer, loss, metrics=None, sample_weight_mode=None)

  • optimizer: 优化器，必须
  • loss: 目标函数，必须
  • metrics: 评价指标，可选
  • sample_weight_mode: 默认”None”按样本赋予权值；”temporal”按时间步赋予权值

训练
fit(self, x, y, batch_size=32, epochs=10, verbose=1, callbacks=None,
validation_split=0.0, validation_data=None, shuffle=True,
class_weight=None, sample_weight=None, initial_epoch=0)

  • x: 单个输入numpy array，多个输入是list of numpy array
  • y: numpy array
  • batch_size: 优化一步的样本数
  • epochs: 训练的总轮数
  • verbose: 日志显示，0不输出，1输出进度条，2每个epoch输出一行记录
  • callbacks: list，其中的元素是Callback函数
  • validation_split: float(0~1)，验证集的比例
  • validation_data: 指定验证集(X,y)，将覆盖validation_split
  • shuffle: 是否随机打乱顺序
  • class_weight: 字典，不同类别映射为不同的权重
  • sample_weight: 样本权重，numpy array
  • initial_epoch: 从指定的epoch开始训练，用于继续之前的训练过程

其他方法

  • add(self, layer): 向模型中添加一个层
  • pop(self): 弹出模型最后的一层，无返回值
  • evaluate(self, x, y, batch_size=32, verbose=1, sample_weight=None): 计算误差
  • predict(self, x, batch_size=32, verbose=0): 预测
  • train_on_batch(self, x, y, class_weight=None, sample_weight=None): 根据一个batch更新参数
  • test_on_batch(self, x, y, sample_weight=None): 在一个batch上评估模型
  • predict_on_batch(self, x): 在一个batch上测试
  • fit_generator(self, generator, steps_per_epoch, epochs=1, verbose=1, callbacks=None, validation_data=None, validation_steps=None, class_weight=None, max_q_size=10, workers=1, pickle_safe=False, initial_epoch=0): 逐个产生batch数据，并用于训练，而不是像fit那样一次将所有数据读入内存
  • evaluate_generator(self, generator, steps, max_q_size=10, workers=1, pickle_safe=False)
  • predict_generator(self, generator, steps, max_q_size=10, workers=1, pickle_safe=False, verbose=0)

噪声层用于给数据附加随机的噪声，用以克服过拟合。
也是一种正则化，仅在训练时使用。

GaussianNoise

添加均值为0，标准差为stddev的高斯噪声。

乘以将均值1，标准差为$\sqrt{\frac{rate}{1-rate}}$ 的噪声。

Keras规范化层BatchNormalization

发表于 2018-05-04
字数统计: 240

在每次迭代的batch里，对上一层的输出规范化。算法分两步：

  1. Normalize: 将输出变为均值0，标准差1；
  2. Scale and Shift: 加入新的均值和方差，避免失去学习到的信息

用公式表示算法流程：
$$\begin{align}
\mu_B &\leftarrow \frac{1}{m} \sum_{i=1}^m x_i &&\text{batch mean} \\
\sigma_B^2 &\leftarrow \frac{1}{m} \sum_{i=1}^m \;(x_i-\mu_B)^2 &&\text{batch variance} \\
\hat{x_i} &\leftarrow \frac{x_i - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}} &&\text{normalize} \\
y_i &\leftarrow \gamma\hat{x_i} + \beta \equiv BN_{\gamma,\beta}\;(x_i) &&\text{scale and shift}
\end{align} $$

代码

  • axis: 指定要规范化的轴
  • momentum: 移动平均和移动方差的冲量，公式如下：
    moving_mean = momentum × moving_mean + (1 - momentum) × batch_mean
    moving_var = momentum × moving_var + (1 - momentum) × batch_var
  • epsilon: 加到方差里的小浮点数，避免除数为0
  • center: 若True，则给规范化后的tensor加上偏移量beta
  • scale: 若True，则乘以gamma

Keras循环层Recurrent

发表于 2018-05-04
字数统计: 714

Recurrent

循环层的基类，所有的循环层(LSTM, GRU, SimpleRNN)都继承本层。

  • cell: RNN cell实例，包括
    • call(input_at_t, states_at_t)方法，返回(output_at_t, states_at_t_plus_1)
    • state_size
  • return_sequences: 默认False返回最后一个输出；若True则返回每个时刻的输出序列
  • return_state: 是否返回最后时刻的隐状态
  • go_backwards: 是否将输入序列逆向处理，即将最后一个当做t=1
  • stateful: 当前batch下标为i的最终状态，将会被用作下个batch下标为i的初始状态
  • unroll: 若True则循环层将被展开，加速计算，但会占用更多内存

SimpleRNN, SimpleRNNCell

一种全连接RNN，输出会回馈到输入。

  • units: 输出维度

GRU, GRUCell

Gated Recurrent Unit有两个变种。
默认实现(1406.1078v3)，reset gate 在矩阵相乘前作用于hidden state；
另一种实现(1406.1078v1)的顺序则相反，和CuDNNGRU (GPU-only) 兼容。
调用时使用’reset_after’=True，’recurrent_activation=’sigmoid’.

LSTM

Long Short Term Memory Layer.

  • unit_forget_bias: 若为true将会在forget gate初始化时加1，并强制 bias_initializer=”zeros”
  • implementatioin: mode 1会使用大量小型计算，mode 2会使用少量大型计算；根据设备与任务选择

ConvLSTM2D

和LSTM类似，但是输入变换和循环变换都是通过卷积完成的。

CuDNNGRU, CuDNNLSTM

Fast GRU/LSTM implementation backed by CuDNN.
只能在GPU上运行，以TensorFlow作为后端。

Keras卷积层Convolutional

发表于 2018-05-04
字数统计: 564

Conv1D, Conv2D, Conv3D

一维卷积层(时域卷积)，用来在一维输入信号上进行邻域滤波。
二维卷积层，对图像的空域卷积。
三维卷积层对三维的输入进行滑动窗卷积。

  • filters: 卷积核的数目
  • kernel_size: 卷积核的长度和宽度
  • strides: 卷积步长；若为单个整数则表示在各个维度步长相同；与dilation_rate不兼容
  • padding: ‘valid’,’same’
  • data_format: ‘channels_first’,’channels_last’，指定通道维的位置
  • dilation_rate: 膨胀比率；与strides不兼容
  • activation: 激活函数，不指定则不使用
  • use_bias: 是否使用偏置项

SeparableCon2D

在深度方向上的可分离卷积，可看做两个卷积核的叠加。
首先对每个channel分别卷积(depthwise)，然后将结果卷积混合到输出(pointwise)。

  • depth_multipiler: 第一步(depthwise)中每个输入channel生成多少个输出channel

Conv2DTranspose

转置卷积层(反卷积)，用于卷积结果的逆向变换。
例如，将具有卷积层输出shape的tensor转换为卷积层输入shape的tensor.

Cropping1D, Cropping2D, Cropping3D

对输入裁剪，cropping=(i,j)指定在每个轴的首、尾分别裁掉多少个元素。

UpSampling1D, UpSampling2D, UpSampling3D

将每个轴的元素分别重复(i,j,k)次。

ZeroPadding1D, ZeroPadding2D, ZeroPadding3D

在数据每个维度上的边界处填充(i,j,k)个0.

Keras常用层Core

发表于 2018-05-03
字数统计: 450

当使用层作为模型首层时，需要指定 input_shape 参数。

Dense

全连接层Dense，实现的运算是 output = activation(dot(input, kernel) + bias).

  • units: 大于0的整数，代表该层的输出维度
  • activation: 如果不指定，则不使用
  • use_bias: 为True则使用偏置项bias

Activation

激活层对一个层的输出施加激活函数。和Dense中的activation作用相同。

Dropout

Dropout在训练过程中每次更新参数时按一定概率随机断开输入神经元。可以防止过拟合。

  • rate: float(0~1)，fraction of the input units to drop
  • noise_shape: shape of the binary dropout mask

Flatten

Flatten层用来将输入展平，将多维数据变成一维，常用在从卷积层到全连接层的过度。
Flatten不会影响batch的大小。

  • data_format: ‘channels_last’, ‘channels_first’

Input

Input() is used to instantiate a Keras tensor.

Reshape

Reshape层用来将输入转换为特定的shape，不包含batch大小。
即Reshape((6,2))将把shape变为(m,6,2)

Permute

Permute(序列改变)层将输入的维度按照给定模式重排顺序，例如将RNN和CNN网络连接时，可能会用到该层。

  • dims: 整数元组，比如(2,1)，将会互换数据的第一、第二维度。

RepeatVector

将输入重复n次。

Lambda

对上一层的输出施加任何表达式

ActivityRegularizer

经过本层的数据不会有任何变化，会更新损失函数

Masking

对输入的信号序列“屏蔽”，来跳过特定的时间步。

  • unit_norm(): 单位模约束，强制矩阵最后一个轴拥有单位模
  • max_norm(m=2): 最大模约束
  • min_max_norm(min_value=0.0, max_value=1.0, rate=1.0, axis=0): 最小/最大模约束

Keras正则项Regularizer

发表于 2018-05-03
字数统计: 180

正则项基于层Layer进行惩罚，惩罚的接口与层有关。
正则函数接受权重矩阵作为输入，计算并返回单个值，添加到目标函数中。

这些层有3个关键字参数可以施加正则项：

  • kernel_regularizer：权重正则
  • bias_regularizer：偏置正则
  • activity_regularizer：输出正则

可用正则项

  • keras.regularizers.l1(0.)
  • keras.regularizers.l2(0.)
  • keras.regularizers.l1_l2(0.)

自定义正则项

任何以权重矩阵作为输入并返回单个数值的函数均可以作为正则项
