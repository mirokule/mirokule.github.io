Q-Q图

发表于 2018-03-21
字数统计: 223

用途

Q-Q图将两个概率分布的分位数放在一起比较，若分布线性相关，则应绘制一条直线。
固定一种为正态分布，则可观察另一套样本是否近似服从正态分布。

作图步骤

  1. 将原始观察数据从小到大排列：$x_1,x_2,…,x_n$
  2. 计算$x$对应的分位频率$p_i=(i-\frac{1}{2})/n,i=1,2,…,n$，这里$P$可以有多种定义。
  3. 计算$p$对应的标准正态分位数$q$:
    $$ P\left{X\leq q_1\right} = \int_{-\infty}^{q_1}\frac{1}{\sqrt{2\pi}}e^{-\frac{x^2}{2}} dx = p_i $$
  4. 将数对$(q_i, x_i)$画在坐标平面上。

因为$q_i$由$p_i$唯一确定，所以若数据$x_i$的分布与正态分布非常接近，则点$(q_i,x_i)$应大致成一条直线。

pandas.get_dummies

发表于 2018-03-14
字数统计: 268

使用pandas.get_dummies()可以对用字符标识的离散特征进行one-hot编码。
只能处理字符类别，不能处理以数字标识的类别

离散特征的编码分为两种情况：

  • 没有大小意义，比如color:{red, blue, green}，使用one-hot编码
  • 有大小意义，比如size:{X, XL, XXL}，使用map映射{X:1, XL:2, XXL:3}

pandas.get_dummies

属性

  • data: Series, DataFrame
  • prefix: 拆分后的列名前缀
  • prefix_sep: 前缀分隔符
  • dummy_na: 默认False，若为True则单独列出NaN
  • columns: 列名，默认None，转换全部列
  • sparse: 默认False，是否返回SparseDataFrame
  • drop_fist: 默认False，是否忽略第一个属性，使k个类别返回k-1个变量

返回

  • dummes: DataFrame, SparseDataFrame

密码学家的工具箱

  • 对称密码
  • 公钥密码
  • 单向散列函数
  • 消息认证码
  • 数字签名
  • 伪随机数生成器

对称密码

对称密码(symmetric cryptography)是指在加密和解密时使用同一种密钥的方式。

1.一次性密码本
随机生成与明文等长的比特序列key，将明文msg与key做异或加密，再做异或解密。
缺点：需要保密配送随机生成的key

2.DES(Data Encryption Standard)
密钥长度56位，每次加密64位信息
3DES：用3个密钥，进行加密->解密->加密过程

3.AES(Rijndael)
由多个轮组成，分组长度128位。

非对称密码

非对称密码(asymmetric cryptography)是指在加密和解密时使用不同密钥的方式，比如公钥密码(public-key cryptography)。
加密使用公开密码，解密使用私有密码，私有密码不需要传输，解决了对称加密中的密钥配送问题。
缺点：处理速度只有对称密码的几百分之一，不适合用来对很长的消息内容加密。

1.RSA
$$ 密文 = (明文)^E\; mod\; N\;\;\;\;\; 明文 = (密文)^D\; mod\; N$$
E和N组成了公钥，用于加密；D和N组成私钥，用于解密。
计算过程：

  • 准备两个很大的质数p和q，相乘得到N
  • 求p-1和q-1的最小公倍数，得到L
  • 找到一个比L小，且和L互质的整数E
  • 找到一个比L小，且(E*D)和L互质的整数D

单向散列函数Hash

根据任意长度的消息计算出固定长度的散列值，特点：

  • 单向性：不能逆推出原信息
  • 抗碰撞(collision)：对同一输出，难以找到两个不同的输入
  • 计算速度快

实例：

  • MD4, MD5
  • SHA-1, SHA-256, SHA-384, SHA-512
  • RIPEMD-160
  • AHS, SHA-3

能识别出文件是否被“篡改”，不能识别出发送者的身份是否“伪装”

消息认证码MAC

消息认证码(message authentication code)是一种确认完整性并进行认证的技术。
输入包括任意长度的消息和双方共享的密钥，输出是固定长度的数据，称为MAC值。

数字签名

同样使用一对钥匙，不过和前文相反，私钥用来加密生成签名，公钥用来解密验证签名。

pandas缺失数据处理

发表于 2018-03-11
字数统计: 384

发现缺失

pandas使用NaN(not a number)表示缺失的数据，使用isnull()
(numpy使用nan表示缺失数据，用np.isnan(np.nan)判断。)

处理缺失

1.删除数据：drop, dropna

参数：

  • labels: 要删除的行列标签
  • index/columns: 行列标签的另一种写法
  • level: 多层索引的层级
  • errors: {‘ignore’, ‘raise’}错误处理
  • axis: 0横向index, 1纵向列
  • how: {‘any’有NA则删, ‘all’全为NA则删}
  • thresh: 阈值，有n个NA时才删除
  • subset: 要考察的子列
  • inplace: 默认False，若为True则修改原数据

返回：

  • drooped(drop): Type of caller
  • dropped(dropna): DataFame
2.填充数据：fillna

参数：

  • value: 填充内容，可以是常量、字典、Series、DataFrame等
  • method: {‘backfill’, ‘bfill’, ‘pad’, ‘ffill’, None}, 使用邻近数据填充
  • axis: 0横向index, 1纵向列
  • inplace: 默认False，若为True则修改原数据
  • limit: 默认None，最大填充长度
  • downcast: 默认None

返回：

  • filled: DataFrame
