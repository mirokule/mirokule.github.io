Keras有两种类型的模型，序贯模型(Sequential)和函数式模型(Functional)。
函数式模型应用更广泛，序贯模型是函数式模型的一种特殊情况。

两类模型有一些通用方法：

  • model.summary(): 输出模型概况，实际调用的是keras.utils.print_summary
  • model.get_config(): 返回包含配置信息的python字典，配置信息可用于重构
  • model.get_layer(): 根据层名称或下标获得层对象
  • model.get_weights(): 返回模型权重，类型为numpy array
  • model.set_weights(): 从numpy array将权重载入给模型
  • model.to_json: 返回代表模型的JSON字符串，只包含网络结构，不包括权值
  • model.to_yaml: 返回代表模型的YAML字符串
  • model.save_weights(filepath): 将权重保存到指定路径，文件类型HDF5(.h5)
  • model.load_weights(filepath, by_name=False): 从HDF5中加载权重

噪声层用于给数据附加随机的噪声，用以克服过拟合。

GaussianNoise

添加均值为0，标准差为stddev的高斯噪声。

Keras规范化层BatchNormalization

发表于 2018-05-04
字数统计: 240

在每次迭代的batch里，对上一层的输出规范化。算法分两步：

  1. Normalize: 将输出变为均值0，标准差1；
  2. Scale and Shift: 加入新的均值和方差，避免失去学习到的信息

用公式表示算法流程：
$$\begin{align}
\mu_B &\leftarrow \frac{1}{m} \sum_{i=1}^m x_i &&\text{batch mean} \\
\sigma_B^2 &\leftarrow \frac{1}{m} \sum_{i=1}^m \;(x_i-\mu_B)^2 &&\text{batch variance} \\
\hat{x_i} &\leftarrow \frac{x_i - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}} &&\text{normalize} \\
y_i &\leftarrow \gamma\hat{x_i} + \beta \equiv BN_{\gamma,\beta}\;(x_i) &&\text{scale and shift}
\end{align} $$

代码

  • axis: 指定要规范化的轴
  • momentum: 移动平均和移动方差的冲量，公式如下：
    moving_mean = momentum × moving_mean + (1 - momentum) × batch_mean
    moving_var = momentum × moving_var + (1 - momentum) × batch_var
  • epsilon: 加到方差里的小浮点数，避免除数为0
  • center: 若True，则给规范化后的tensor加上偏移量beta
  • scale: 若True，则乘以gamma

Keras循环层Recurrent

发表于 2018-05-04
字数统计: 714

Recurrent

循环层的基类，所有的循环层(LSTM, GRU, SimpleRNN)都继承本层。

  • cell: RNN cell实例，包括
    • call(input_at_t, states_at_t)方法，返回(output_at_t, states_at_t_plus_1)
    • state_size
  • return_sequences: 默认False返回最后一个输出；若True则返回每个时刻的输出序列
  • return_state: 是否返回最后时刻的隐状态
  • go_backwards: 是否将输入序列逆向处理，即将最后一个当做t=1
  • stateful: 当前batch下标为i的最终状态，将会被用作下个batch下标为i的初始状态
  • unroll: 若True则循环层将被展开，加速计算，但会占用更多内存

SimpleRNN, SimpleRNNCell

一种全连接RNN，输出会回馈到输入。

  • units: 输出维度

GRU, GRUCell

Gated Recurrent Unit有两个变种。
默认实现(1406.1078v3)，reset gate 在矩阵相乘前作用于hidden state；
另一种实现(1406.1078v1)的顺序则相反，和CuDNNGRU (GPU-only) 兼容。
调用时使用’reset_after’=True，’recurrent_activation=’sigmoid’.

LSTM

Long Short Term Memory Layer.

  • unit_forget_bias: 若为true将会在forget gate初始化时加1，并强制 bias_initializer=”zeros”
  • implementatioin: mode 1会使用大量小型计算，mode 2会使用少量大型计算；根据设备与任务选择

ConvLSTM2D

和LSTM类似，但是输入变换和循环变换都是通过卷积完成的。

CuDNNGRU, CuDNNLSTM

Fast GRU/LSTM implementation backed by CuDNN.
只能在GPU上运行，以TensorFlow作为后端。

Keras卷积层Convolutional

发表于 2018-05-04
字数统计: 564

Conv1D, Conv2D, Conv3D

一维卷积层(时域卷积)，用来在一维输入信号上进行邻域滤波。
二维卷积层，对图像的空域卷积。
三维卷积层对三维的输入进行滑动窗卷积。

  • filters: 卷积核的数目
  • kernel_size: 卷积核的长度和宽度
  • strides: 卷积步长；若为单个整数则表示在各个维度步长相同；与dilation_rate不兼容
  • padding: ‘valid’,’same’
  • data_format: ‘channels_first’,’channels_last’，指定通道维的位置
  • dilation_rate: 膨胀比率；与strides不兼容
  • activation: 激活函数，不指定则不使用
  • use_bias: 是否使用偏置项

SeparableCon2D

在深度方向上的可分离卷积，可看做两个卷积核的叠加。
首先对每个channel分别卷积(depthwise)，然后将结果卷积混合到输出(pointwise)。

  • depth_multipiler: 第一步(depthwise)中每个输入channel生成多少个输出channel

Conv2DTranspose

转置卷积层(反卷积)，用于卷积结果的逆向变换。
例如，将具有卷积层输出shape的tensor转换为卷积层输入shape的tensor.

Cropping1D, Cropping2D, Cropping3D

对输入裁剪，cropping=(i,j)指定在每个轴的首、尾分别裁掉多少个元素。

UpSampling1D, UpSampling2D, UpSampling3D

将每个轴的元素分别重复(i,j,k)次。

ZeroPadding1D, ZeroPadding2D, ZeroPadding3D

在数据每个维度上的边界处填充(i,j,k)个0.

Keras常用层Core

发表于 2018-05-03
字数统计: 450

当使用层作为模型首层时，需要指定 input_shape 参数。

Dense

全连接层Dense，实现的运算是 output = activation(dot(input, kernel) + bias).

  • units: 大于0的整数，代表该层的输出维度
  • activation: 如果不指定，则不使用
  • use_bias: 为True则使用偏置项bias

Activation

激活层对一个层的输出施加激活函数。和Dense中的activation作用相同。

Dropout

Dropout在训练过程中每次更新参数时按一定概率随机断开输入神经元。可以防止过拟合。

  • rate: float(0~1)，fraction of the input units to drop
  • noise_shape: shape of the binary dropout mask

Flatten

Flatten层用来将输入展平，将多维数据变成一维，常用在从卷积层到全连接层的过度。
Flatten不会影响batch的大小。

  • data_format: ‘channels_last’, ‘channels_first’

Input

Input() is used to instantiate a Keras tensor.

Reshape

Reshape层用来将输入转换为特定的shape，不包含batch大小。
即Reshape((6,2))将把shape变为(m,6,2)

Permute

Permute(序列改变)层将输入的维度按照给定模式重排顺序，例如将RNN和CNN网络连接时，可能会用到该层。

  • dims: 整数元组，比如(2,1)，将会互换数据的第一、第二维度。

RepeatVector

将输入重复n次。

Lambda

对上一层的输出施加任何表达式

ActivityRegularizer

经过本层的数据不会有任何变化，会更新损失函数

Masking

对输入的信号序列“屏蔽”，来跳过特定的时间步。

使用后端框架的激活函数

高级激活函数

对于后端Theano/TensorFlow/CNTK不能表达的复杂激活函数，如PReLU, LeakyReLU等，可以通过高级激活层(Advanced Activation)实现。
注意此时是在构建layer对象。

LeakyReLU层

LeakyReLU是ReLU的特殊版本，当不激活时仍有非零输出值，从而获得一个小梯度，避免神经元“死亡”现象。
$$f(x)=
\begin{cases}
\alpha\cdot x& \text{x < 0}\\
x& \text{x >= 0}
\end{cases}$$

参数alpha: float >= 0，代表第三象限中线段的斜率。

PReLU层

参数化的ReLU(Parametric ReLU)，表达式为：
$$f(x)=
\begin{cases}
\alpha\cdot x& \text{x < 0}\\
x& \text{x >= 0}
\end{cases}$$

其中参数 alpha 是通过学习得到的，和X具有相同形状。

  • alpha_initializer: alpha的初始化函数
  • alpha_regularizer: alpha的正则项
  • alpha_constraint: alpha的约束项
  • shared_axes: 该参数指定的轴将共享同一组参数
ELU层

指数线性单元(Exponential Linear Unit)，表达式为：
$$f(x)=
\begin{cases}
\alpha(e^x - 1)& \text{x < 0}\\
x& \text{x >= 0}
\end{cases}$$

ThresholdedReLU层

带有门限的ReLU，表达式为：
$$f(x)=
\begin{cases}
x& \text{x > theta}\\
0& \text{otherwise}
\end{cases}$$

Keras初始化方法Initializer

发表于 2018-05-02
字数统计: 603

初始化方法定义了对Keras层设置初始化权重的方法。
不同的层可能使用不同的关键字，一般是 kernel_initializer 和 bias_initializer.

Initializer

Initializer的基类，不能直接使用，所有的Initializer均继承该类。

Zeros, Ones, Constant

初始化为指定的固定值0, 1 或者 value

Identity, Orthogonal

Identity：使用单位矩阵初始化，仅适用2D方阵。gain是单位矩阵的Multiplicative Factor
Orthogonal：使用正交矩阵初始化。

RandomNormal, RandomUniform

Random Normal: 正态分布初始化，参数(均值，标准差，随机种子)
Random Uniform: 均匀分布初始化，参数(下界，上界，随机种子)

TruncatedNormal

Truncated(截尾) Normal的不同之处在于，与均值相差2个标准差以上的点会被舍弃。
推荐神经网络的权重使用。

VarianceScaling

该初始方法能够自适应目标张量的shape。

  • distribution=’normal’: Truncated normal, mean=0, std=$\sqrt{\frac{scale}{n}}$
  • distribution=’uniform’: [-limit, limit]的均匀分布，其中$limit=\sqrt{\frac{3\times scale}{n}}$

其中，n从mode选择的’fan_in’, ‘fan_out’, ‘fan_avg’中选取；scale是缩放因子。

lecun_normal, lecun_uniform

LeCun normal: Truncated normal, mean=0, std=$\sqrt{\frac{1}{fan_{in}}}$
LeCun uniform: [-limit, limit]的均匀分布，其中$limit = \sqrt{\frac{3}{fan_{in}}}$
$fan_{in}$是权重tensor的输入单元数。

he_normal, he_uniform

He normal: Truncated normal, mean=0, std=$\sqrt{\frac{2}{fan_{in}}}$
He uniform: [-limit, limit]的均匀分布，其中$limit = \sqrt{\frac{6}{fan_{in}}}$
$fan_{in}$是权重tensor的输入单元数。

glorot_normal, glorot_uniform

又称作 Xavier Normal Initializer 和 Xavier Uniform Initializer.
glorot normal: Truncated normal, mean=0, std=$\sqrt{\frac{2}{fan_{in}+fan_{out}}}$
glorot normal: [-limit, limit]的均匀分布，其中$limit = \sqrt{\frac{6}{fan_{in}+fan_{out}}}$
$fan_{in}$是权重tensor的输入单元数，$fan_{out}$是输出单元数。

自定义初始化方法

初始化方法必须是callable的，并且接受 shape 和 dtype 两个参数，返回符合要求的张量。

假设空间H作用于数据量为N的样本集D，h的数量看上去是无穷的，但真正有效(effective)的h的数量确实有限的，这个数量记做$m_H(N)$.
H中每一个h作用于D都能算出一个$E_{in}$来，共有$m_H(N)$个不同的$E_{in}$

Dichotomy

二分类问题中给定样本集D，如果D是线性可分的，就把一种分类方式称作一个Dichotomy(对分)。
一个Dichotomy中包含了所有可以分出这个局面的h，相当于把假设空间H里的所有h按照不同的分类结果来分组。
样本集D上的所有划分组成了dichotomy set.

Growth Function

对特定样本数N，可以有多个采样集D，每个D对应不同的dichotomy set。
将这些dichotomy set的最大值，定义为N的成长函数Growth Function: m(N)。
增长函数表示假设空间H对D中样本赋予标记的最大可能结果数。

Shatter

Shatter(打散)指假设空间H能够实现D中全部示例的对分，即增长函数=$2^N$
另一种定义：对于一个假设空间H，如果存在m个数据样本能够被假设空间H中的函数按所有可能的$2^h$种形式分开，则称假设空间H能够把m个数据样本打散。

Break Point

使Growth Function < $2^H$成立的N的最小值。
若k是Break Point，那么k+1, k+2, …对应的Growth Function都会小于$2^N$

VC Bound

将成长函数的上界，记为B(N,k)，意为：maximum possible m_H(N) when break point = k.

  • B(2,2) = 3
  • B(3,2) = 4
  • B(N,1) = 1
  • B(N,k) = $2^N$ for N<k; $2^N-1$ for N=k

最后利用数学归纳法，可以证明得到下面的bounding function(N>k)
$$ m_H(N) \leq \sum_{i=0}^{k-1} \binom{N}{i} $$


如果假设空间存在有限的break point，那么$m_H(2N)$会被最高幂次为k-1的多项式上界给约束住。
随着N的逐渐增大，指数式的下降会比多项式的增长更快，所以此时VCBound是有限的。
更深的意义在于，N足够大时，对H中的任意一个假设h，$E_{in}(h)$都将接近于$E_{out}(h)$，这表示机器学习可行的第一个条件是有可能成立的。

Vapink-Chervonenkis Dimension

假设空间H的VC维是能被H打散的最大的样本集的大小，记做$d_{vc}(H)$。
若对任意数目的数据样本都有函数能将它们shatter，则假设空间的VC维为无穷大。
VC维的大小只与模型和假设空间有关，反应了假设空间H的强大程度(powerfulness)。VC维越大，H越强，能打散更多的点。

根据定义，可以得到一个明显的结论：
$$ k = d_{vc}(H) + 1$$

一个实践规律：VC维与假设参数w的自由变量数目大约相等。
模型较复杂($d_vc$较大)时，需要更多的训练数据。实际经验里，一般$N\approx 10\times d_{vc}$
