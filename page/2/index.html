Box-Cox变换

使用Box-Cox变换一般都可以成功地数据正态变换，但在二分变量或较少水平的等级变量的情况下，变换不能成功。
变换后，残差可以更好地满足正态性、独立性等假设前提，降低伪回归的概率。

$$ y = \frac{x^\lambda - 1}{\lambda}\;\;for\;\;\lambda > 0$$
$$ y = Log(x) \;\; for \;\; \lambda = 0 $$

相关函数

1.scipy.stats.boxcox(x, lmbda=None, alpha=None)

参数

  • x: 输入序列，要求一维、正数
  • lmbda: 若为None，则自动寻找maximizes the log-likelihood function的参数$\lambda$
  • alpha: 若不为None，则计算100*(1-alpha)%置信区间

返回

  • boxcox: 转换后的ndarray
  • maxlog: 输入lmbda为None时的自适应参数
  • (min_ci, max_ci): 给定alpha的置信区间
2.scipy.special.boxcox(x, lmbda)
  • 若$x<0$则返回nan.
  • 若$x==0,\lambda<0$返回-inf

$$ y = \frac{x^\lambda - 1}{\lambda}\;\;for\;\;\lambda != 0$$
$$ y = Log(x) \;\; for \;\; \lambda = 0 $$

参数

  • x: array_like
  • lmbda: 参数

返回

  • y: array
3.scipy.special.boxcox1p(x, lambda)

返回(1+x)的box-cox变换，通过平移解决$x=0$的边界情况

  • 若$x<-1$则返回nan
  • 若$x==-1, \lambda<0$则返回-inf

$$ y = \frac{(1+x)^\lambda - 1}{\lambda}\;\; for \;\; \lambda != 0 $$
$$ y = Log(1+x)\;\; for \;\; \lambda = 0 $$

参数

  • x: array-like
  • lmbda: 参数

返回

  • y: array

sklearn.OneHotEncoder

发表于 2018-03-21
字数统计: 238

LabelEncoder将不同类别转换为数字，比如[‘paris’,’tokyo’]转为[2,3]，引入了类别大小属性。
OneHotEncoder将类别编码成m个二值类别，只标示类别，没有引入大小。
比get_dummies的优势在于可以处理数字类别。

定义

class sklearn.preprocessing.OneHotEncoder(n_values='auto',categorical_features='all',dtype=<class'numpy.float64'>,sparse=True,handle_unknown='error')

参数

  • n_values: 特征的类别数量，’auto’由训练数据计算，也可以指定int
  • categorical_features: 指定要转换的字段
  • dtype: 输出的类型
  • sparse: 默认True返回系数矩阵，若为False返回Array
  • handle_unknown: 是否raise error

属性

  • active_features_: 训练集中的被转换的values
  • feature_indices_: indices to feature ranges
  • n_values_: 每个特征的number of values

方法

  • fit(X, y=None)
  • fit_transform(X, y=None)
  • transform(X)
  • get_params([deep])
  • set_params(**params)

Q-Q图

发表于 2018-03-21
字数统计: 223

用途

Q-Q图将两个概率分布的分位数放在一起比较，若分布线性相关，则应绘制一条直线。
固定一种为正态分布，则可观察另一套样本是否近似服从正态分布。

作图步骤

  1. 将原始观察数据从小到大排列：$x_1,x_2,…,x_n$
  2. 计算$x$对应的分位频率$p_i=(i-\frac{1}{2})/n,i=1,2,…,n$，这里$P$可以有多种定义。
  3. 计算$p$对应的标准正态分位数$q$:
    $$ P\left{X\leq q_1\right} = \int_{-\infty}^{q_1}\frac{1}{\sqrt{2\pi}}e^{-\frac{x^2}{2}} dx = p_i $$
  4. 将数对$(q_i, x_i)$画在坐标平面上。

因为$q_i$由$p_i$唯一确定，所以若数据$x_i$的分布与正态分布非常接近，则点$(q_i,x_i)$应大致成一条直线。

pandas.get_dummies

发表于 2018-03-14
字数统计: 268

使用pandas.get_dummies()可以对用字符标识的离散特征进行one-hot编码。
只能处理字符类别，不能处理以数字标识的类别

离散特征的编码分为两种情况：

  • 没有大小意义，比如color:{red, blue, green}，使用one-hot编码
  • 有大小意义，比如size:{X, XL, XXL}，使用map映射{X:1, XL:2, XXL:3}

pandas.get_dummies

属性

  • data: Series, DataFrame
  • prefix: 拆分后的列名前缀
  • prefix_sep: 前缀分隔符
  • dummy_na: 默认False，若为True则单独列出NaN
  • columns: 列名，默认None，转换全部列
  • sparse: 默认False，是否返回SparseDataFrame
  • drop_fist: 默认False，是否忽略第一个属性，使k个类别返回k-1个变量

返回

  • dummes: DataFrame, SparseDataFrame
