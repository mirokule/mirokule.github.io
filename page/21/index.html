Presenting actionsheet on iPad

Error

The modalPresentationStyle of a UIAlertController with this style is UIModalPresentationPopover. You must provide location information for this popover through the alert controller’s popoverPresentationController. You must provide either a sourceView and sourceRect or a barButtonItem. If this information is not known when you present the alert controller, you may provide it in the UIPopoverPresentationControllerDelegate method -prepareForPopoverPresentation.’

Solve

if sender is a UIBarButtonItem

if let popoverController = alertController.popoverPresentationController {
    popoverController.barButtonItem = sender
}

else

if let popoverController = alertController.popoverPresentationController {
    popoverController.sourceView = exchangeButton
    popoverController.sourceRect = exchangeButton.bounds
    popoverController.permittedArrowDirections = UIPopoverArrowDirection.Up
}

Insert

i 在当前位置前插入
I 在当前行首插入
a 在当前位置后插入
A 在当前行尾插入
o 在当前行之后插入一行
O 在当前行之前插入一行

Move

h 左移一个字符
l 右移一个字符
k 上移一个字符
j 下移一个字符

w 向前移动一个单词，光标停在单词首部
b 向后移动一个单词，光标停在单词首部
e，同w，光标停在单词尾部
ge，同b，光标停在单词尾部

^ 移动到本行第一个非空白字符上
0（数字0）移动到本行第一个字符上

移动到本行第一个字符。同0健。
$ 移动到行尾 3$ 移动到下面3行的行尾

gg 移动到文件头。 = [[
G（shift + g） 移动到文件尾。 = ]]

f（find）命令也可以用于移动，fx将找到光标后第一个为x的字符，3fd将找到第三个为d的字符。
F 同f，反向查找。

:n 跳到指定行，比如跳到240行就是 :240回车
nG 跳到指定行，比如230G跳到230行

Ctrl + e 向下滚动一行
Ctrl + y 向上滚动一行
Ctrl + d 向下滚动半屏
Ctrl + u 向上滚动半屏
Ctrl + f 向下滚动一屏
Ctrl + b 向上滚动一屏

Undo

u 撤销（Undo）
U 撤销对整行的操作
Ctrl + r 重做（Redo）

Delete

x 删除当前字符
X 删除当前字符的前一个字符
dd 删除当前行
10d 删除当前行开始的10行

D 删除当前字符至行尾。D=d$
d$ 删除本行当前字符之后的所有字符

:1,10d 删除1-10行
:11,$d 删除11行及以后所有的行

J(shift + j)　删除两行之间的空行

Copy

yy 拷贝当前行
nyy 拷贝当前行之后的n行
ndd 剪切当前行之后的n行
:1,10d 将1-10行剪切

p 在当前光标后粘贴,如果之前使用了yy命令来复制一行，那么就在当前行的下一行粘贴。
shift+p 在当前行前粘贴

:1,10 co 20 将1-10行复制到第20行之后
:1,10 m 20 将1-10行移动到第20行之后
:1,$ co $ 将整个文件复制一份并添加到文件尾部

Quit

:wq 保存并退出
ZZ 保存并退出
:q! 强制退出并忽略所有更改
:e! 放弃所有修改，并打开原来文件
