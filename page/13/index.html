Rotation

旋转本质上需要知道 1个向量 + 1个角度，共4个值。有3种表示方法：

  • Matrix(矩阵): 使用 4*4 矩阵，共16个值，计算时有额外的时间和空间消耗；

  • Euler(欧拉): 使用3个值(xRotation, yRotation, zRotation)，是一系列坐标轴旋转的组合，按顺序沿各个轴旋转；会带来万向节锁问题。
    Gimbal Lock(万向节锁)：当欧拉旋转的第二次旋转角度为90°时，第三个轴就被转到了第一个轴相同的方向，因此丢失了一个自由度。

  • Quaternion(四元数): 本质上是一个高阶复数，即 x = a + bi + cj + dk，效率更高

    rigidBody.rotation = Quaternion.Euler(0.0f, 0.0f, rigidBody.velocity.x * -1);

乘法

  1. quaternion quaternio，例如q = q1 q2，表示将一个点先q1旋转，再q2旋转
  2. quaternion Vector3，例如p = q v，表示将向量v进行q旋转，得到旋转后的向量p.
    Quaternion必须位于乘号的左边。
//镜头到物体的距离
distance = Vector3.forward * 10.0f;
transform.position = target.position - distance
//有旋转
transform.position = target.position - rotation * distance

GKRandom

Randomization Distributions

  • Random: GKRandomDistribution(:)
  • Gaussian: GKGausianDistribution(:), good for making corridors
  • Shuffled: GKShuffledDistribution(:), does not repeat a number until all the numbers have been returned, good for making rooms

Randomization Sources

  • Linear Congruential: GKLinearCongruentialRandomSource(), least computation and randomness
  • ARC4 random: GKARC4RandomSource(), the default source
  • Mersenne Twister: GKMersenneTwisterRandomSource(), most computation and randomness

GKGraph is a collection of locations in game scene, which are used in path finding. Subclass of GKGraph includes:

  • GKGridGraph: A list of valid locations on a two-dimensional grid. Entities are only allowed to travel between these grid locations. Example as Pac-Man.

  • GKObstacleGraph: A collection of invalid polygons that specify the shapes of the obstacles in the scene.

Type

There are two kinds of types in swift: named types and compound types.

#Named Type

A named type is a type that can be given a particular name when it is defined.
Named types include classes, structures, enumerations, protocols and arrays, dictionaries, optional values.

Compound Type

A compound type is a type without a name: function types and tuple types.

Type Casting

Type casting is a way to check the type of an instance, or to treat that instance as a different superclass or subclass.
Type casting is implemented with the is and as operators.
