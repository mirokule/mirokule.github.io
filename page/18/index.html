UITableView Pull-to-Refresh

发表于 2015-08-11
字数统计: 60

ViewDidLoad

In viewDidLoad(), add the following code:

self.refreshControl = UIRefreshControl()
self.refreshControl?.backgroundColor = UIColor.purpleColor()
self.refreshControl?.tintColor = UIColor.whiteColor()
self.refreshControl?.addTarget(self, action: Selector("actionHandler"), forControlEvents: UIControlEvents.ValueChanged)

When the tableview is pulled down sufficiently, the refresh controll triggers the UIControlEvents.ValueChanged.

ActionHandler

It’s your responsibility to hide the refresh control.

func actionHandler() {
    ...
    self.refreshControl?.endRefreshing()
}
