SceneKit Force and Torque

发表于 2016-05-01
字数统计: 70

Force

Can affect linear and angular momentum.

    let force = SCNVector3(x: randomX, y: randomY, z: 0)
    let position = SCNVector3(x: 0.05, y: 0.05, z: 0.05)
geometryNode.physicsBody?.applyForce(force, atPosition: position, impulse: true)

Torque

Only affects angular momentum.

let torque = SCNVector4(x: 0.1, y: 0.1, z: 0.1, w: 1.0)

The w-componet determines the rotation angle, that is, the magnitude of the torque.

SceneKit Geometry Node

发表于 2016-05-01
字数统计: 51

3D primitives in SceneKit: cone, torus, capsule, tube, pyramid, box, sphere, cylinder.

var geometry: SCNGeometry = SCNSphere(radius: 1.0)

let geometryNode = SCNNode(geometry: geometry)
geometryNode.physicsBody = SCNPhysicsBody(type: .Dynamic, shape: nil)

scnScene.rootNode.addChildNode(geometryNode)

If shape of physicsBody is nil, Scene Kit will generate shape according to boundings of geometry.

SCNCamera

字数统计: 40

Set up a camera with SCNCamera, attach it to a SCNNode, and then add to the rootNode of the scene.

func setupCamera() {
    cameraNode = SCNNode()
    cameraNode.camera = SCNCamera()
    cameraNode.position = SCNVector3(x: 0, y: 2, z: 10)

    scnScene.rootNode.addChildNode(cameraNode)
}

SCNView

Consider SCNView something like skView. SCNView contains SCNScene and can display various information.

scnView.showsStatistics = true
scnView.allowsCameraControl = true
scnView.autoenablesDefaultLighting = true

scnView.scene = scnScene

SCNScene

SCNScene is something like skScene. SCNScene has a rootNode which should be parent node of other nodes in the scene.

scnScene.background.contents = "GeometryFighter.scnassets/Textures/Background_Diffuse.png"
scnScene.rootNode.addChildNode(cameraNode)

Color

发表于 2015-11-18
字数统计: 137

Elements of color

  1. HSB，对应眼睛感知
  • Hue: 色调、色相，光的波长
  • Saturation: 饱和度，鲜艳度
  • Brightness: 明度、亮度，明暗
  1. RGB，光的三原色，加色混合，叠加为白
  • Red: 红
  • Green: 绿
  • Blue: 蓝

取值：0~255

  1. CMY，色(颜料)的三原色，减色混合，叠加为黑
  • Cyan: 青 = 蓝+绿
  • Magenta: 品红 = 蓝+红
  • Yellow: 黄 = 红+绿

取值： 0~100%

由于印刷工艺颜色不纯，引入黑色，代码K(与Blue区分)，组成CMYK
CMYK与RGB色域不完全相同，有些颜色可显示不可印刷

CGAffineTransform

发表于 2015-11-04
字数统计: 133

The CGAffineTransform data structure represents a MATRIX used for affine transformations.
A transformation specifies how points in one coordinate system map to points in another coordinate system.

Creating

  • CGAffineTransformMakeRotation(_ angle: CGFloat) -> CGAffineTransform
  • CGAffineTransformMakeScale(_ sx: CGFloat, _ sy: CGFloat) -> CGAffineTransform
  • CGAffineTransformMakeTranslation(_ tx: CGFloat, _ tx: CGFloat) -> CGAffineTransform: move by axis

Modification

  • CGAffineTransformTranslate(_ t: CGAffineTransform, _ tx: CGFloat, _ ty: CGFloat) -> CGAffineTransform: move the transform
  • CGAffineTransformScale
  • CGAffineTransformRotate
  • CGAffineTransformInvert(_ t: CGAffineTransform) -> CGAffineTransform
  • CGAffineTransformConcat(_ t1: CGAffineTransform, _ t2: CGAffineTransform) -> CGAffineTransform: combine two transform

Apply

  • CGRectApplyAffineTransform(_ rect: CGRect, _ t: CGAffineTransform) -> CGRect: apply the transform to rectangle

Evaluating

  • CGAffineTransformIsIdentity(- t: CGAffineTransform) -> Bool
  • CGAffineTransformEqualToTransform(_ t1: CGAffineTransform, _ t2: CGAffineTransform) -> Bool
