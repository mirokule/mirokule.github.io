NumPy

发表于 2016-05-25
字数统计: 992

NumPy(Numerical Python)是高性能科学计算和数据分析的基础包。它是高级工具的构建基础。
NumPy最重要的特点是它的N维数组对象ndarray.
NumPy的运算是基于数组的，不同于Math基于数字。
Numpy的设计目的是处理大数据，默认传递引用，而不是复制数据。

ndarray

ndarray是同种元素的多维数组，快速而灵活的大数据集容器。

通用函数

条件逻辑(numpy.where)

numpy.where()是三元表达式 x if condition else y的矢量化版本。

随机数生成(numpy.random)

numpy和Python中都有random包，但是实现功能不同。
python的random函数生成一个数，而numpy中的random函数可以生成数组。

统计函数

mean、std等函数既可以作为数组的实例方法(A.mean())，又可以作为Numpy顶级函数调用(np.mean(A))，包括：

  • sum(),mean()
  • std(),var()
  • min(),max()
  • argmin(),argmax()
  • cumsum(),cumprod()

其中，布尔值会被强行转为1和0，所以sum()可以用来对布尔数组中的真值计数。
