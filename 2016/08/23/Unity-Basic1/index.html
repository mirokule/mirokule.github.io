Mesh 网格
Mesh Filter 网格过滤器，用于获取网格并传递给对象
Mesh Render 网格渲染器，从网格过滤器获得网格，在scene视图中渲染，添加Material
Skinned Mesh Render 蒙皮渲染器，用于呈现骨骼动画

Prefab 预设体，游戏对象及组件的组合，目的是使对象可重复利用
Shuiken 粒子系统
Mecanim 人形动画系统

Camera
Perspective 透视镜头，锥体镜头范围，远小近大
Orthographic 正交镜头，长方体镜头范围，远近大小相同

Physics
1.Rigidbody 刚体，用于接受外力
Drag：阻力
Damper：阻尼，数值越大物体运动越慢
Torque：转矩

  1. Collider 碰撞体，用于碰撞检测
    Is Trigger：触发器，选中该项则可用于触发事件，忽略物理碰撞

3.Hinge Joint：铰链关节，由两个刚体组成，适用于模拟链和钟摆等
Character Joint：角色关节，扩展的球关节，限制关节在不同旋转轴下的旋转角度

Audio Listener 音频检测器，接收输入和场景中的任何声音
Audio Source 音频源，随距离衰减
Audio Clip 音频剪辑
Output 输出，混音器
Bypass Effect 直通？简单的打开/关闭所有音效
Bypass Listener Effect 忽略Listener上应用的效果
Bypass Reverb Zone 忽略混响区域
Pitch 音调
Stereo Pan 立体效果
Spatial Blend 空间混合
Reverb Zone Mix 对混响区域的混合系数
Doppler level 多普勒级别
Volume Rolloff 衰减模式
Audio Reverb Zone 音频混响

Light Probes 光照探头，对某个位置的光照信息采样
NavMesh 导航网格，生成可自动导航的路径
Occlusion Culling 遮挡剔除，当一个物体被其他物体遮挡住时不进行渲染
Frustum Culling 视锥体剔除，不渲染摄像机视锥范围之外的物体

Flare 炫光
Bloom 泛光，多物体眩光
Tonemapping 色调映射，模拟人眼适应环境明暗变化的效果
Antialiasing 抗锯齿，平滑图像
Screen Space Ambient Occlusion 屏幕空间环境遮挡，实时模拟真实的全局光漫反射效果
Twirl 扭曲特效
Vortex 漩涡特效

Develop
Cross 叉乘，外积，左手定则，得到向量，新向量垂直于原来的两个向量，长度等于原向量长度相乘后再乘夹角的正弦
Dot 点乘，内积，数量积，得到标量，数值等于两个向量长度相乘后再乘以二者夹角的余弦
Certain Events 必然事件，Unity中一定条件下自动调用的函数，包括Awake, Start, Update, FixedUpdate, LateUpdate
C#只有内建数组，没有Array数组，但可以使用List容器来达到同样的目的。List不支持下标[i]，只能add(), sort(), foreach()
MonoBehaviour 所有Unity脚本的父类，定义了基本事件
GameObject.Find(“ObjectName”) 返回指定名称的游戏对象的引用
object.GetComponent() 返回游戏对象的组件，如Transform, Rigidbody, Renderer, Light, Camera, Collider, Animation, Audio
由于耗时，避免在Update中使用Find，在Start中Find并保存，或者声明变量后在Inspector中指定
常用API：Transform, Time, Random, Mathf, Coroutine(协同程序)
C#中，协同函数的返回类型必须为IEnumerator，yield要用yield return替代，启动协同程序要使用StartCoroutine函数

Input
Unity中所有输入信息的更新都在Update方法中完成
Input.GetMouseButtonDown(1) Input.GetKeyDown(KeyCode.Space)
Touch 触摸操作返回的结构体

Canvas 画布，每个GUI控件都是画布的子对象
Canvas Group 对UI元素分组
Rect Transform 矩形变换，表示一个可容纳UI元素的矩形，调整时不影响字体大小，不同于Transform的缩放
Anchor Presets 锚点预置按钮
Pivot 轴心点，旋转和缩放都围绕轴心点发生变化
Transition Options 过渡选项，不同状态时的不同颜色、图像或者动画
Toggle 开关勾选
Layout Element 布局元素

Shader 着色器
ShaderLab Unity提供的用来编写Shader的语言
Surface Shaders 表面着色器
Vertex and Fragment Shaders 顶点和片段着色器
Fixed Function Shaders 固定功能管线着色器，一般用于不支持高级着色器特性的旧硬件上
Self-Illuminated Shader Family 用于有自发光效果的对象
Reflective Shader Family 用于能反射环境立方体贴图的不透明对象
Unlit 仅使用纹理颜色，不受光照影响
VertextLit 顶点光照
Diffuse 漫反射
Specular 在漫反射的基础上增加高光计算
Normal Mapped 法线贴图
Normal Mapped Specular 带高光法线贴图
Parallax Normal Mapped 视差法线贴图
Parallax Normal Mapped Specular 带高光视差法线贴图

StandardShader 标准着色器，5.0新引入，推荐使用
Albedo 反射率
Occlusion 散射
Emission 自发光
Tiling 平铺
Smoothness 光滑度，不能和 Metallic 同时使用
Conservation of Energy 能量守恒原理

Reflection Probe 反射探头，用于捕获周围环境的光反射信息，将捕获的图像存储为Cubemap和能在游戏对象上使用的反射材质。非常消耗资源

AssetBundle 在游戏运行时对资源进行动态的下载和加载
HLAPI - High Level API - Network Manager 简单易上手
LLAPI - Low Level API - Network Transport API 强大而灵活
Remote Procedure Calls 远程过程调用, RPC
Profiler 性能分析
