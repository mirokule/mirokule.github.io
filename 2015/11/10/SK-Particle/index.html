SKEmitterNode

SKEmitterNode can be generated from code or sks files.

Target Node: The initial properties of new particles are based on the emitter, but in future frames, the particles are treated as children of the target node.

SKKeyframeSequence

An SKKeyframeSequence object specifies values for keyframes used in emitter simulation.

  • particleColorSequence
  • particleColorBlendFactorSequence
  • particleScaleSequence
  • particleAlphaSequence

InterpolationMode

The interpolationMode property specifies how to calculate the values for times between each keyframe.

  • Linear: default
  • Spline: 样条插值，生成平滑曲线
  • Step: does not interpolate, value as that of the most recent keyframe

RepeatMode

The repeatMode property specifies how to calculate values if they’re outside of the keyframes defined in the sequence.
Example: sequence defines [0.5, 1.0], then [0, 0.5] is calculated with repeatMode.

  • Clamp: default, [0, 0.5] will be the same as 0.5
  • Loop: [0, 0.5] will loop as [0.5, 1.0]
